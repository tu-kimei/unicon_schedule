Luồng vận chuyển Container

Xác nhận 2 luồng hàng Xuất và Nhập

Hàng Xuất (Export)

Export Flow
1. Lấy cont rỗng
📸 Chụp trong/ngoài
→
2. Tới kho đóng hàng
📸 Phiếu vào/ra kho, cân xe
→
3. Chở ra cảng
📸 Phiếu vào/ra cảng
→
4. Xong ✓

Hàng Nhập (Import)

Import Flow
1. Lấy cont có hàng từ cảng
📸 Phiếu ra cảng
→
2. Chở tới kho xuống hàng
📸 Phiếu vào/ra kho, cân xe
→
3. Trả cont rỗng
📸 Phiếu vào cảng/bãi
→
4. Xong ✓
